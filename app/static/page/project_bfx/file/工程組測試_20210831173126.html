0818討論¶

  • 一個光碟會有多張照片，判斷那個部位(約20幾個部位)最嚴重，並呈現其部位嚴重程度(無、龜裂、不完全骨折、骨折)

  • label的部分、程式的部分

  • 資料預處理 論文 標註

  • ＣＮＮ的架構

# filename:流水號、CODED_PART:部位代號、CODED_LEVEL:嚴重程度代號、PART:部位名稱、LEVEL:嚴重程度名稱、BodyPartExamined:身體位置 # RescaleSlope:(可略)、RescaleIntercept:(可略)、IS_NEG:是否為負片（背景是為白色）、size:圖片大小、CODED_ID:、CODED_APLY：受編
